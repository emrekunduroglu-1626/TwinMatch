# TwinMatch Mobile

Android öncelikli React Native/Expo proje iskeleti ve auth ekranları.

## Kapsam

- Expo tabanlı React Native başlangıç yapısı
- React Navigation native stack
- Zustand auth store
- Auth akışı:
  - Splash
  - Onboarding
  - Register
  - OTP Verification
  - Age Confirm
  - Login

## Çalıştırma

```bash
npm install
npm run start
```

## Sonraki adımlar

- `services/` altında gerçek API katmanı
- `SetupStack` ve `MainTab` navigasyonu
- Form validasyonu ve kalıcı token saklama
