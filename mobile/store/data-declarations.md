# Veri Beyan Formları — Birebir Kopyalanabilir Yanıtlar

## Google Play "Data safety"
| Soru | Yanıt |
|---|---|
| Veri topluyor mu? | Evet |
| Toplanan: Kişisel bilgiler | Ad, e-posta, telefon, doğum tarihi |
| Toplanan: Fotoğraf/Video | Evet (profil + doğrulama) |
| Toplanan: Konum | Yaklaşık + Hassas (yalnızca kullanım anında, mekan önerisi) |
| Toplanan: Mali bilgiler | Satın alma geçmişi (kart verisi HAYIR) |
| Toplanan: Mesajlar | Uygulama içi mesajlar (ikiz sohbetleri) |
| Üçüncü tarafla paylaşım | Hayır (işleyiciler hariç: ödeme/SMS) |
| Aktarımda şifreleme | Evet (TLS) |
| Silme talebi mekanizması | Evet (uygulama içi hesap silme) |
