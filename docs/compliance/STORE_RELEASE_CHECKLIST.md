# Store Release Checklist

## Kod tarafı
- [x] Backend auth gerçek JWT ile çalışıyor.
- [x] Hediye Freemium kilidi backend seviyesinde var.
- [x] Hediye notu PII moderasyonu var.
- [x] Adres kasası doğrulaması olmadan kabul engeli var.
- [x] Kullanıcı engelleme/şikayet endpointleri var.
- [x] Hesap silme/veri dışa aktarma endpointleri var.
- [x] Mobile audit 0 vulnerability.
- [x] Admin audit 0 vulnerability.
- [x] Admin mock fallback UAT/production'da kapatılabilir.

- [ ] App Store Connect uygulama kaydı.
- [ ] Google Play Console uygulama kaydı.
- [ ] Privacy Policy ve Terms hukukçu onayı.
- [ ] Google Play Data Safety formu.
- [ ] App Store Privacy Nutrition Labels.
- [ ] Google Maps API anahtarı.
- [ ] SMS sağlayıcı canlı hesabı.
- [ ] Ödeme sağlayıcı canlı hesabı.
- [ ] Amazon/Çiçeksepeti tedarikçi/affiliate/ordering sözleşmeleri.
- [ ] TestFlight ve Google Play Internal Testing gerçek cihaz onayı.
